О проекте Nova Git

Nova Git — это демонстрационный интерфейс, показывающий, как можно интегрировать локальные или облачные ИИ‑модули для генерации контента, кода и обучающих примеров прямо в браузере. Система способна адаптироваться под контекст пользователя и обеспечивать мгновенную обратную связь без серверной части.

  • Полностью автономный интерфейс (работает оффлайн).
  • Поддержка русских и английских запросов.
  • Расширяемая архитектура: можно подключить OpenAI API или локальную LLM.
